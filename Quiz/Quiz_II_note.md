    ### *4. Preliminary operations of “Queues”*
    ### <font color = "red">My ans:</font>
    ```
    top() 取最前端的值
    pop() 刪除最前端的值
    push() 將值加入最後端
    ```
    ### <font color = "green">Correct ans:</font>
    ```
    enqueue(): Add an element to the rear of the queue.
    dequeue(): Remove an element from the front of the queue.
    front(): Access the front element without removing it.
    isEmpty(): Check whether the queue is empty.
    isFull(): Check whether the queue is full (only for array-based queues).
    ```
